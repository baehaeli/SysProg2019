# SysProg2019

시스템 프로그래밍 쉘 구현 과제

명령어 실행 방법 및 기능
1. ls : 현재 디렉토리에 있는 파일 목록을 보여준다.
2. cd

  -cd .. : 상위 디렉토리로 이동
  
  -cd : 홈 디렉토리로 이동
  
  -cd 경로 : 경로로 이동
  
3. exit : 쉘을 종료한다.
4. mkdir : 새로운 디렉토리를 만든다
  -mkdir a/b/c : 현재 디렉토리에 a디렉토리를 만들고 a 디렉토리에 b, b 디렉토리에 c 디렉토리를 만든다.
5. rm : 파일을 삭제한다.
6. pwd : 현재 디렉토리를 알려준다.
